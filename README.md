# stress_level_prediction

Machine learning project to predict stress levels using a self-collected dataset and Random Forest algorithm, with a complete data processing and training pipeline.

## Files

- `load_dataset.py`: Loads the dataset
- `preprocess.py`: Preprocesses the data
- `split_data.py`: Splits the data into train/test sets
- `train.py`: Trains the model
- `evaluate.py`: Evaluates the model
- `main.py`: Main script to run the pipeline
- `stress_dataset.csv`: The dataset file

## Usage

1. Install dependencies: `pip install -r requirements.txt` (if exists)
2. Run `python main.py`

## Requirements

- Python 3.x
- Libraries: pandas, scikit-learn, etc. (add as needed)
